# UC-D2 — الحجز مع الفرز (Booking + Triage) · ثلاث محادثات اختبارية

تغطي فضاء قرار الفرز: ثقة عالية، حالة غامضة، واختيار المتصل المسبق. الصيغة
رسمية ("أنتم"). كل محادثة تنتهي ببوابة تأكيد ("هل أنتم متأكدون؟ → نعم")، ثم
`book_visit` برمز تأكيد حقيقي، وقراءة رقم الحجز حرفاً حرفاً (الصيغة `DV-XXXXX`)،
ثم سؤال رضا العميل. تُستدعى الأدوات بأسمائها الحقيقية ومساراتها.

---

## 1) مسار الثقة العالية — القلب

**CALLER:** مرحباً، عندي ألم في الصدر وضيق نفس من يومين وبدّي أحجز موعد.

**AGENT (سلمى):** سلامتكم. لتأكيد هويتكم، ممكن رقم هاتفكم رقماً رقماً من فضلكم؟

**CALLER:** صفر، سبعة، تسعة، واحد، صفر، صفر، صفر، صفر، صفر، واحد.

**AGENT (سلمى):** شكراً، رقمكم ينتهي بـ صفر واحد، صحيح؟
// → calls get_patient_by_phone({ phone: "+962791000001" })

**CALLER:** صحيح.

**AGENT (سلمى):** بناءً على وصفكم لألم الصدر وضيق النفس، يبدو أن عيادة القلب هي
الأنسب. هل تودّون أن أحجز لكم فيها؟
// → calls suggest_specialty({ chief_complaint: "ألم في الصدر وضيق نفس" })  // top_confidence ≈ 0.85 → عيادة القلب

**CALLER:** نعم، عيادة القلب.

**AGENT (سلمى):** لدينا مواعيد متاحة الأسبوع القادم. أقرب موعد يوم الثلاثاء
العاشرة صباحاً. هل يناسبكم؟
// → calls check_availability({ specialty_id, from_date, to_date })

**CALLER:** نعم يناسبني.

**AGENT (سلمى):** للتأكيد: حجز في عيادة القلب يوم الثلاثاء العاشرة صباحاً. هل
أنتم متأكدون؟

**CALLER:** نعم.

**AGENT (سلمى):** تم الحجز. رقم حجزكم هو: دي‑في، شرطة، سبعة، كاف، اثنان، قاف، ميم
— أي `DV-7K2QM`. احتفظوا به من فضلكم.
// → calls book_visit({ spoken_phone, specialty_id, chief_complaint, picked_by: "bot_suggestion", start_time, confirmation_token })

**AGENT (سلمى):** كيف تقيّمون سهولة المكالمة من ١ إلى ١٠؟
// → calls submit_ces_rating({ conversation_id })

**CALLER:** تسعة. شكراً.

**AGENT (سلمى):** شكراً لكم وسلامتكم. مع السلامة.

**AGENT (سلمى):** سلامتكم. ممكن رقم هاتفكم رقماً رقماً للتأكد من هويتكم؟

**CALLER:** صفر، سبعة، ثمانية، واحد، صفر، صفر، صفر، صفر، صفر، اثنان.

**AGENT (سلمى):** شكراً لكم.
// → calls get_patient_by_phone({ phone: "+962781000002" })

// → calls suggest_specialty({ chief_complaint: "تعب وصداع متواصل" })  // باطنية ≈ 0.5، أنف وأذن ≈ 0.5

// → calls submit_ces_rating({ conversation_id })

**AGENT (سلمى):** شكراً لكم، مع السلامة.

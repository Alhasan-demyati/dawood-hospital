# UC-D3 — تعديل / إلغاء / حالة الموعد · ثلاث محادثات اختبارية

تغطّي إعادة الجدولة، والإلغاء، والاستعلام عن الحالة فقط. الصيغة رسمية ("أنتم").
أدوات الكتابة (`reschedule_visit` / `cancel_visit`) لا تُستدعى إلا بعد بوابة
تأكيد صريحة ورمز تأكيد حقيقي. المحادثة الثالثة لا تستدعي أي أداة كتابة.

---

## 1) إعادة جدولة — القلب

**CALLER:** مرحباً، عندي موعد في عيادة القلب وبدّي أأجّله.

**AGENT (سلمى):** بكل سرور. ممكن رقم هاتفكم رقماً رقماً للتأكد من هويتكم؟

**CALLER:** صفر، سبعة، تسعة، واحد، صفر، صفر، صفر، صفر، صفر، واحد.

**AGENT (سلمى):** شكراً. لديكم موعد في عيادة القلب يوم الثلاثاء العاشرة صباحاً.
هل هذا الموعد الذي تودّون تأجيله؟
// → calls get_patient_by_phone({ phone: "+962791000001" })
// → calls list_visits({ patient_id })

**CALLER:** نعم، بدّي أأجّله للأسبوع الجاي.

**AGENT (سلمى):** متوفر يوم الثلاثاء القادم الحادية عشرة صباحاً، أو الأربعاء
التاسعة صباحاً. أيّهما تفضّلون؟
// → calls check_availability({ specialty_id: "<cardiology>", from_date, to_date })

**CALLER:** الثلاثاء الحادية عشرة.

**AGENT (سلمى):** للتأكيد: نقل موعدكم إلى الثلاثاء القادم الحادية عشرة صباحاً.
هل أنتم متأكدون؟

**CALLER:** نعم.

**AGENT (سلمى):** تم تعديل الموعد بنجاح إلى الثلاثاء الحادية عشرة صباحاً، ورقم
الحجز كما هو. هل أستطيع مساعدتكم بشيء آخر؟
// → calls reschedule_visit({ booking_reference, new_start, confirmation_token })

**CALLER:** لا، شكراً.

**AGENT (سلمى):** كيف تقيّمون سهولة المكالمة من ١ إلى ١٠؟
// → calls submit_ces_rating({ conversation_id })

**CALLER:** تسعة.

**AGENT (سلمى):** شكراً لكم، مع السلامة.

## 2) إلغاء — الجلدية

**CALLER:** السلام عليكم، بدّي ألغي موعدي في عيادة الجلدية.

**AGENT (سلمى):** وعليكم السلام. ممكن رقم هاتفكم رقماً رقماً من فضلكم؟

**CALLER:** صفر، سبعة، ثمانية، واحد، صفر، صفر، صفر، صفر، صفر، اثنان.

**AGENT (سلمى):** شكراً. لديكم موعد في عيادة الجلدية يوم السبت الحادية عشرة
صباحاً. هل تودّون إلغاءه؟
// → calls get_patient_by_phone({ phone: "+962781000002" })
// → calls list_visits({ patient_id })

**CALLER:** نعم، بدّي ألغيه.

**AGENT (سلمى):** للتأكيد: إلغاء موعد عيادة الجلدية يوم السبت الحادية عشرة
صباحاً. هل أنتم متأكدون؟

**CALLER:** نعم.

**AGENT (سلمى):** تم إلغاء الموعد. إن احتجتم الحجز مجدداً نحن في خدمتكم. هل
أستطيع مساعدتكم بشيء آخر؟
// → calls cancel_visit({ booking_reference, reason: "بطلب المريض", confirmation_token })

**CALLER:** لا، شكراً.

**AGENT (سلمى):** كيف تقيّمون سهولة المكالمة من ١ إلى ١٠؟
// → calls submit_ces_rating({ conversation_id })

**CALLER:** ثمانية.

**AGENT (سلمى):** شكراً لكم، مع السلامة.

## 3) الاستعلام عن الحالة فقط — الأسنان

**CALLER:** مرحباً، بس بدّي أتأكد من تاريخ ووقت موعدي في عيادة الأسنان.

**AGENT (سلمى):** بكل سرور. ممكن رقم هاتفكم رقماً رقماً من فضلكم؟

**CALLER:** صفر، سبعة، سبعة، واحد، صفر، صفر، صفر، صفر، صفر، ثلاثة.

**AGENT (سلمى):** شكراً. موعدكم في عيادة الأسنان يوم الأحد التاسعة والنصف
صباحاً، ورقم الحجز ينتهي بـ "تي كي". هل من شيء آخر تودّون تعديله؟
// → calls get_patient_by_phone({ phone: "+962771000003" })
// → calls list_visits({ patient_id })
// ملاحظة: استعلام فقط — لا تُستدعى أي أداة كتابة.

**CALLER:** لا، هذا كل شي. شكراً.

**AGENT (سلمى):** كيف تقيّمون سهولة المكالمة من ١ إلى ١٠؟
// → calls submit_ces_rating({ conversation_id })

**CALLER:** تسعة.

**AGENT (سلمى):** شكراً لكم، نراكم في موعدكم. مع السلامة.
